# Angular Event Timetable

A responsive event scheduling application built with Angular 19 and Angular Material. Features a weekly timetable view with multi-venue event management, horizontal/vertical scrolling, and persistent data storage.

## Features

- **Weekly Timetable View**: Navigate between weeks with arrow buttons
- **Multi-Venue Support**: Events can span across multiple venues
- **Dual Scrolling System**: Vertical time slots (8 AM - 6 PM) and horizontal venue columns
- **Event Management**: Color-coded events with time and venue positioning
- **Local Storage Persistence**: Saves venues, events, selected day, and week navigation
- **Responsive Design**: Fixed venue columns (200px) with synchronized scrolling
- **Material UI Integration**: Clean interface with Angular Material components

## Tech Stack

- Angular 19.2.19
- Angular Material
- TypeScript
- CSS Grid/Flexbox
- Local Storage API

## Quick Start

```bash
ng serve
```

Navigate to `http://localhost:4200/` to view the timetable.
